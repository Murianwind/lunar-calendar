🚀 구글 API 발급 및 사용 가이드

이 서비스는 개인의 Google Calendar API를 사용하여 작동합니다. 안전한 이용을 위해 본인만의 인증 정보(API Key, Client ID)를 직접 발급받아야 합니다.

1단계: Google Cloud 프로젝트 생성

1. Google Cloud Console에 접속합니다.
2. 상단 프로젝트 선택창에서 [새 프로젝트]를 클릭하여 이름을 입력하고 생성합니다.

2단계: API 활성화

1. 왼쪽 메뉴에서 [API 및 서비스] > [라이브러리]로 이동합니다.
2. Google Calendar API를 검색하고 [사용] 버튼을 클릭합니다.

3단계: OAuth 동의 화면 설정 (중요)

1. [OAuth 동의 화면] 메뉴에서 User Type을 외부(External)로 선택합니다.
2. 앱 이름과 이메일 등 필수 정보를 입력하고 저장합니다.
3. [테스트 사용자] 섹션에서 본인의 구글 이메일을 반드시 추가해야 로그인이 가능합니다.

4단계: API 키 및 클라이언트 ID 발급

1. [사용자 인증 정보] 메뉴로 이동합니다.
2. [사용자 인증 정보 만들기] > [API 키]를 생성하여 복사합니다.
3. [사용자 인증 정보 만들기] > [OAuth 클라이언트 ID]를 클릭합니다.
4. 유형은 웹 애플리케이션을 선택하고, 승인된 JavaScript 원본에 현재 홈페이지 주소(예: https://본인아이디.github.io)를 추가합니다.
5. 생성된 클라이언트 ID를 복사합니다.

5단계: 서비스 이용

복사한 ID와 Key를 메인 화면에 입력하고 [인증정보 저장]을 누른 뒤 사용하세요!
